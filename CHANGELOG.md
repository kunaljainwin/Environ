# Changelog

All notable changes to the **Environ** extension will be documented here.

---

## [0.0.2] - 2025-01-01
### Added
- Logo fixed.

## [0.0.1] - 2025-01-01
### Added
- Initial release of Environ extension.
- System Information panel showing OS, CPU, memory, and kernel.
- Installed tool version scanner with smart detection.
- Environment Variables viewer with search.
- Snapshot creation (system, versions, environment).
- Snapshot comparison with color-coded diff viewer.
- Sidebar TreeView showing system, versions, env vars, and snapshots.
- Refresh command for Sidebar View.
- Security-safe HTML Webview UI.
